# CodeAlpha Tasks

This repository contains the tasks completed as part of the CodeAlpha UI/UX Internship.

## Task 1 - Adidas Store Wireframe

For this task, I created a low-fidelity wireframe for an Adidas Store mobile application using Figma.

The wireframe includes:

* Login Screen
* Home Screen
* Product Details Screen
* Cart Screen

The main focus was on creating a simple user flow and page layout before adding colors and visual elements.
