# CodeAlpha Tasks

This repository contains the tasks completed as part of the CodeAlpha UI/UX Internship.

## Task 3 - Zomato UX Case Study

For this task, I analyzed the user experience of the Zomato application.

The case study includes:

* User Journey Analysis
* Strengths and Weaknesses
* Suggested Improvements
* Simple Wireframe Sketches for the proposed features

The goal was to understand how users interact with the app and suggest ways to improve their experience.
