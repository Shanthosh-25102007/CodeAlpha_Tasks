# Adidas Store Mobile App Design

This project was completed as part of a Graphics Designing internship task.

## Project Overview

The objective of this project was to design a mobile e-commerce application for Adidas. The project started with low-fidelity wireframes and was later converted into an interactive prototype.

## Features

The application includes:

* Login Screen
* Home Screen
* Product Details Screen
* Cart Screen

## Design Process

1. Created low-fidelity wireframes to define the layout and user flow.
2. Organized content and navigation between screens.
3. Added interactive connections between screens.
4. Developed a clickable prototype using Figma.

## User Flow

Login → Home → Product Details → Cart

The prototype demonstrates how users can browse products, view product details, and add items to their cart.

## Tools Used

* Figma

## Deliverables

* Low-Fidelity Wireframes
* Clickable Prototype
* User Flow Design
